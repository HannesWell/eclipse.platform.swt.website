Eclipse 3.5 - New and Noteworthy in SWT

  • Milestone M1
  • Milestone M2
  • Milestone M3
  • Milestone M4

Milestone M1
Cairo on AIX Cairo support has been added to the AIX platform. This enables advanced graphics support on AIX.

Milestone M2
JNI Generator A new plugin based tool has been added to simplify the process of generating the SWT JNI code. For more info click here.

Milestone M3
SWT for 32-bit Cocoa 32-bit OSX Cocoa has joined the lineup of supported configurations for which SWT and Eclipse are routinely built. This implementation is currently early access. Everyone is encouraged to download and test these new builds, and to log bugs in bugzilla.

Block selection in StyledText StyledTexts can be switched into block selection mode with new API StyledText.setBlockSelection(true).

Milestone M4
SWT for 64-bit Cocoa 64-bit OSX Cocoa is now available. As with the 32-bit implementation, this work is still under active development, but available as an early access release. Everyone is encouraged to download and test these new builds, and to log bugs in bugzilla.
Javascript-to-Java invocations Javascript can now call out to a Java-side function that has been created on a Browser instance. Values of type number, boolean, string, null, array and exception can be passed between the two sides (example snippet).
Improved Java-to-Javascript invocations Java can now evaluate a Javascript string in the context of a Browser document and receive a return value of type number, boolean, string, null, array, or exception (example snippet).
New Caret Listener New API has been added to StyledText to allow you to listen for changes in the caret offset. See StyledText.addCaretListener() and StyledText.removeCaretListener() for more information.
New Text Editor Demo Eric Hung of the University of Waterloo has contributed a new Text Editor demo application. The application makes extensive use of StyledText API's to demonstrate how they can be used to create a full featured text editor.
